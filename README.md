Hydra colorscheme is based on [onedarkpro](olimorris/onedarkpro.nvim) and [github-theme](projekt0n/github-nvim-theme)

## Installation

```vim
Plug 'maxmx03/hydra.nvim', { 'branch': 'vim' }
```

## Configuration

`set termguicolors` is enabled by default

```vim
colorscheme hydra
```

## Commands

`Hydra`
